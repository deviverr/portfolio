A mobile-first Solana marketplace with USDC escrow, local trading, Next.js frontend work, Anchor program code, and Supabase-backed data flows. This is the clearest signal that I can think in product, frontend, and crypto mechanics at the same time.

solana-resilience-sdk

TypeScript / Solana SDK

A Solana SDK focused on RPC and transaction reliability. It shows comfort with client infrastructure, failover thinking, CLI tooling, tests, and developer experience around Solana apps.

raydium-cpmm-arb-monitor

TypeScript / Raydium / Solana

A real-time monitor for Raydium CPMM pools using Raydium SDK and @solana/web3.js. It is backend-ish, data-heavy, and useful proof for building interfaces that need live market or protocol state.

FPV-Motor-Live-Speaker

JavaScript / Web Audio / Web Serial

A vanilla browser app that captures audio, detects dominant frequency, and sends motor commands to a Betaflight flight controller over USB. It shows I can build media-heavy, real-time browser UI without hiding behind a framework.
