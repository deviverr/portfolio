# deviverr Portfolio

Minimal static GitHub Pages portfolio.

Live site:

```text
https://deviverr.github.io/portfolio/
```

Local preview:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Notes

- No build step or framework required.
- Text-only page: HTML and CSS only.
- Content is limited to public repository names, descriptions, README/package metadata, visible language data, and contact links.
